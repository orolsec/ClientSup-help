Режим работы: Стандартный API (PostgREST)

Тип аутентификации: Anon Public

Уровень безопасности: RLS (Row Level Security) необходимо активировать для всех таблиц

Рекомендуемые политики:

Read: SELECT (anon)

Write: INSERT (anon) / UPDATE (authenticated)
